Menu

The menu contains functions to load/save a theme project.

Theme Tree

The theme tree displays all elements of the loaded theme.

The icons infront of every node indicate the type of the entry:
  • An <include> node. Includes another theme XML file.
  • An <images> node. This node is used to load PNG files and declare image nodes.
    • An <area> node. This is the basic element from with all others are constructed.
    • An <alias> node. References other image nodes and can modify some aspects.
    • An <grid> node. Used to assemble grids with flexible resizing weights.
    • An <composed> node. Used to layer multiple images on top of each other.
    • An <select> node. Renders one of the child images based on conditions.
    • An <animation> node. Annimations can cycle through different images and optionally adjust the tint or size.
    • An <cursor> node. Defines a mouse cursor.
    • An <cursorref> node. Creates an mouse cursor alias.
  • An <theme> node. Contains images and parameters for widgets.
  • An <fontdef> node. Loads a font and defines font parameters.
Elements can be inserted, deleted and moved through a context menu or using the buttons below:
  • Moves the select node up.
  • Moves the select node down.
  • Deletes the selected node. When it is still referenced by other nodes then these will turn red indicating an error.
  • Inserts a new child node into the selected node.
  • Clones the selected node. The clone will be inserted below and it's name is altered either by appending a number or if the name ends in a number it is incremented.
  • Copies the selected node into the system clipboard.

Theme Properties

The theme properties panel displays the properties for the currently select theme node.

Image preview

Displays the image when the selected node is an image node. When an <area> or <cursor> node is selected then this pane also allows to edit the coordinates by dragging the rulers.

The preview currently can't display complex image nodes (like <alias>, <grid>, <composed>, <select> or <animation>)
